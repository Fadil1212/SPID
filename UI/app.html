Sports Celebrity Image Classification Project


Project Objective:

The Sports Celebrity Image Classification project aims to accurately classify images of famous sports personalities from around the world. The project focuses on identifying the following sports celebrities: Cristiano Ronaldo, Kobe Bryant, Lionel Messi, Maria Sharapova, Roger Federer, Serena Williams, and Virat Kohli.


Project Details:

In this project, an SVM (Support Vector Machine) classifier was utilized to train the model. The SVM algorithm is well-known for its effectiveness in image classification tasks. Through extensive training and testing, the model achieved an impressive accuracy of 82%.


Dataset Information:

The dataset used in this project consists of images categorized into seven classes, each representing one of the featured sports celebrities. The dataset contains a diverse collection of over 3000 images, ensuring a robust and comprehensive training experience.
